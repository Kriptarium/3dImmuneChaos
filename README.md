
# 3dImmuneChaos

Streamlit app supporting **multi-channel (DAPI / Immune / Phase)** organoid imaging preprocessing + **chaos metrics** (Lyapunov & Fractal Dimension) and **PDF/CSV reporting**.

## Files to upload to GitHub (drag & drop)
- `app.py`
- `preprocess_core.py`
- `chaos_metrics.py`
- `report_utils.py`
- `requirements.txt`

## Run locally
```
pip install -r requirements.txt
streamlit run app.py
```
